Caesar Cipher

Now let's learn about Caesar Cipher!

Imagine you have a secret message that you want to send to your friend. What if someone else steals and reads your secret message? How can your messages be protected?

How does Caesar Cipher work?

Caesar Cipher works by shifting the alphabet set by a certain number of positions. You can do this by choosing a special key, then using that to find the corresponding letter, and repeat this to encrypt the whole message.

Caesar cipher is a way to encode the message so only the people with the special key can access the original message called the plaintext.
This enciphering method is named after Roman leader Julius Caesar, who used it to send secret messages to his army in the field - without protecting his messages there is a greater risk of the opponent stealing and finding out his whole plan!

Try these simple steps to encode your message!

1. Type in the key you want (e.g. 3)

The key determines how many positions the letters of an alphabet set will shift. For example, if the key is 3, each letter will move 3 positions to the right, wrapping around at the end of the alphabet. For example, B becomes E (and A becomes X, etc).

2. Type in the plaintext message you want to encipher in the left textbox (e.g. "My name is __")

(Tip: Type slowly too see the letter from the plaintext highlight in red with the corresponding ciphertext highlight in yellow!)

3. The enciphered message will appear in the right textbox

Check if you get the same encoding by finding the letter you want to encipher on the grey inner-wheel, which represents the original alphabet set (or the plaintext), then replacing it with the corresponding letter on the light blue outer-wheel, which represents the 4 shifted alphabet set (or the ciphertext). Repeat this until you get the enciphered text yourself!

Challenge/food for thought: What happens if you set the key to be bigger than 26 (e.g. 29)? What do you notice?
